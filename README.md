## ⚡ Overview

This project is a minimal, embeddable countdown designed for:

- Notion dashboards
- Personal mission tracking
- High-performance goal execution
- Aesthetic hacker/cyberpunk setups

It displays:

- Days
- Hours
- Minutes
- Seconds

All in a **dark hacker interface** with neon green accents.

---

## 🧠 Features

- ⏳ Real-time countdown
- 🧩 Custom target name
- 📅 Custom date & time
- 💾 Local storage (saves automatically)
- 🖥️ Clean embeddable UI
- 🟢 Hacker aesthetic (green terminal style)
